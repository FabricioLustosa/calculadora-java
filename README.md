**Calculadora Java - Console**

Projeto de calculadora intertiva via terminal desenvolvido
 em Java puro como parte da minha trilha de estudos para construção de 
portfólio.

Sobre o Projeto

Uma calculadora simples de linha de comando que realiza as
 quatro operações matemáticas básicas com tratamento de erros e loop de 
repetição, permitindo múltiplos cálculos em uma única execução.

Funcionalidades

- Soma, subtração, multiplicação e divisão
- Suporte a números decimais
- Validação de entradas inválidas (letras, caracteres especiais)
- Tratamento de divisão por zero
- Loop de repetição: calcula novamente sem reiniciar o programa
- Resultado formatado com 2 casas decimais

Tecnologias utilizadas

- Java SE17+
- java.util.Scanner -> leitura de entrada do usuário
- java.util.Locale -> padronização de formato numérico

Conceitos praticados

- Orientação a objetos e estrutura de classes
- Estruturas de controle: switch, while, continue, break
- Tratamento de excessões com try-catch
- Formatação de saída com printf e especificadores de formato
- Limpeza de buffer do Scanner apóes entradas inválidas
